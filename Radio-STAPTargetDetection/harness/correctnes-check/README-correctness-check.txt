This folder contains images of plots obtained using the reference implementation 
from Matlab Phased Array Toolkit. 

Plots included here are for medium input case. Figures STAP_Medium_Figure01.png 
through STAP_Medium_Figure11.png provide visual information on STAP scene 
formulated using the input generator. Reference solutions using DPCA and 
Advanced-DPCA from Matlab toolkit are included in the plot image files.
